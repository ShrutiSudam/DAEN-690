Location-wise User List

This is the list of users sorted based on the weighted score for each location. It helps to identify the users with the highest adverse impact on the system, at each location so that they can be trained at the earliest possible.


Column names and description:

  • LOCATION: LOCATION ID.
  • USER_ID: ID of the user making erroneous transactions.
  • WEIGHTED_SCORE: Weighted score of a user.
